✨ About Our Magical WebServ ✨

A journey into the wonderful world of web servers! 🌐✨

WebServ is our magical creation - a web server built from scratch in C++98! But what exactly is a web server? Great question! 🤔

🍽️ Imagine Your Website is Like a Restaurant! 🍽️

🏢 The Web Server (WebServ) is like the restaurant building - it's always open, waiting for hungry customers!

👨‍🍳 The Chef is like our CGI scripts (Python, PHP, Perl, Ruby) - they cook up special dishes (dynamic content) just for you!

📋 The Menu is like your HTML pages - it shows you all the yummy things you can order!

🎂 The Kitchen Storage is like our file system - where we keep all the ingredients (images, documents, files)!

🚚 The Delivery Service is like HTTP methods - different ways to get your food delivered!

🤓 Cool Technical Facts!

🔧 Built in C++98: We used the classic version for maximum compatibility!

⚡ Non-blocking I/O: Our server can handle multiple customers at once!

🍪 Cookie Support: We remember your preferences (like your magical theme)!

📝 CGI Scripts: Dynamic content in Python, PHP, Perl, and Ruby!

📁 File Uploads: Share your beautiful photos with the world!

🎨 Custom Error Pages: Even our errors are cute!

💭 Why Did We Build This Magical Server?

At 42 School, we believe in learning by doing! 🎓
Instead of just using existing web servers, we built our own to understand exactly how the internet works under the hood. It's like building your own car to really understand how engines work! 🚗✨
